# MiniLang Programming Language

A mini programming language with a complete compiler implementation in Python using PLY (Python Lex-Yacc).

## Features

- ✅ Arithmetic expressions with proper precedence (PEMDAS)
- ✅ Variables and identifiers
- ✅ String literals and output statements
- ✅ Control flow: if-else, while loops, for loops with step
- ✅ Logical operators: and, or, not
- ✅ Relational operators: ==, !=, <, >, <=, >=
- ✅ Exception handling: try-catch blocks
- ✅ Block statements with begin/end
- ✅ Comments (-- style)
- ✅ Proper variable scoping
- ✅ Comprehensive error handling
